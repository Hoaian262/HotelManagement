## Hotel Management
Ứng dụng Hotel Management đơn giản viết bằng C# (Console) để quản lý phòng, khách và đặt phòng.
## Chức năng
1. sơ đồ khách sạn
2. Thông tin phòng
3. đặt phòng
4. thống kê doanh thu
5. danh sách đặt phòng
6. quản lý dữ liệu
7. lưu file
## Yêu cầu
1. Hệ điều hành Window hoặc MacOS
2. Đã cài .NET SDK 9.0 trở lên
3. Đã cài đặt Visual Studio hoặc Visual Studio Code 
## Cài đặt & sử dụng 
1. Clone the repository
   git clone https://github.com/Hoaian262/HotelManagement.git
2. Mở folder trong VS hoặc VS code
3. Chạy chương trình bằng cách nhấn F5 hoặc
   dotnet restore
   dotnet build
   dotnet run
## Ghi chú
Project mang tính học tập, phục vụ cho môn Cơ sở lập trình
Không sử dụng cơ sở dữ liệu (database)
Code được viết đơn giản, dễ hiểu
